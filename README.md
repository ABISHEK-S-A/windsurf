# Text Summarization Tool

A GUI-based text summarization tool that helps you quickly summarize large chunks of text.

## Features

- Paste or upload text files
- Customizable summary length
- Clean and intuitive interface
- Real-time summarization
- Editable summary output

## Installation

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python text_summarizer.py
```

## Usage

1. Launch the application
2. Either paste text directly into the input area or upload a text file
3. Adjust the summary length using the spinner
4. Click "Generate Summary" to create a concise summary
5. The summary will appear in the output area and can be edited if needed

## Requirements

- Python 3.7+
- PyQt5
- NLTK
- Spacy
